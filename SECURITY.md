# Security Policy

## Supported Versions
Security fixes are provided for the latest commit on `main`.

## Reporting a Vulnerability
Do not open public issues for security reports.

Please report vulnerabilities privately by either:
- GitHub Security Advisories (preferred): `Security` tab -> `Report a vulnerability`
- Email: `keitarou.kondou@gmail.com`

Include:
- Affected version/commit
- Reproduction steps or proof-of-concept
- Impact assessment
- Suggested mitigation (if available)

We will acknowledge receipt and triage as quickly as possible.
